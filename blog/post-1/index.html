AVAYA 発信の仕組み

By Themefisher 08 February 2020

Avayaの発信の仕組みは非常に複雑である。今回は、発信の全体概要を説明し、次回以降で詳細を説明する。

発信の仕組み　- 概要 -

コールセンターから発信する場合、オペレータが電話機又はソフトフォンから架電操作を行うとAvaya Aura Communication Managerにて処理を行い、Media-GatewayからPSTN(公衆交換電話網)を通して、架電先へ着信する。では、この発信操作を行う際に、コールセ ンター内のPBXではどのような処理を行っているのだろうか。

発信業務を行う際にオペレータが実際に行う操作は、

  1. 受話器をあげるまたはソフトフォンの発信ボタンをプッシュする
  2. 電話機またはソフトフォンを操作し、 0 + 電話番号 をプッシュ
  3. 発信音(プルルルル)が鳴り、相手の番号に着信する

のたった3つの操作である。

では、PBXではどのようなどのような処理がなされるのか簡単に説明する。

  1. オペレータが受話器が上げるとトーン(プープープー)を返す
  2. 0 + 電話番号の先頭の「0」を入力することで外線発信の処理開始
  3. 電話機が所属するPBX内のグループと0 + 電話番号の電話番号の数値から発信ルートを確定
  4. PBXに収容されている電話回線からPSTNに到達
  5. PSTNを通って相手の電話機に着信

かなり簡単に書いたが、実際にはもっと複雑である。 特に2～4の処理が非常に複雑なので、より詳細な説明は次回以降で行う。

発信に関わるパラメータ

発信処理に関連するPBX内のパラメータはどのようなものがあるだろうか. 以下に一通り列挙してみる。

  • ars analysis

  • trunk-group
